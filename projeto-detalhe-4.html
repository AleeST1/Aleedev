Gestão de Veículos

Python MySQL
Data: Junho de 2025

Descrição

Este é um sistema completo de gestão de veículos desenvolvido em Python para rodar no terminal (CMD), que gerencia veículos, manutenções, IPVAs e multas. O projeto foi desenvolvido com foco em usabilidade, robustez e escalabilidade, enfrentando diversos desafios técnicos e de negócio.

Demonstração em Vídeo

Funcionalidades

  • FUNCIONALIDADE 1: ✅ Gestão completa de veículos (CRUD);
  • FUNCIONALIDADE 2: ✅ Controle de manutenções com tipos e status;
  • FUNCIONALIDADE 3: ✅ Gestão de IPVA com cálculos automáticos;
  • FUNCIONALIDADE 4: ✅ Sistema de multas com detalhamento;
  • FUNCIONALIDADE 5: ✅ Relatórios avançados e exportação
  • FUNCIONALIDADE 6: ✅ Sistema de usuários e permissões
  • FUNCIONALIDADE 7: ✅ Configurações personalizáveis
  • FUNCIONALIDADE 8: ✅ Interface colorida e intuitiva

Desafios e Soluções

Desenvolver um sistema completo de gestão de veículos em Python para terminal que integrasse múltiplas funcionalidades (veículos, manutenções, IPVA, multas) com interface intuitiva, banco de dados robusto e distribuição standalone.
