Sistema de Gerenciamento de Estoque

JavaScript HTML CSS MySQL
Data: Abril de 2025

Descrição

Este projeto foi desenvolvido para gerenciar o estoque de uma empresa, permitindo adicionar, editar e excluir produtos, além de exportar os dados para Excel e buscar por produtos.

O sistema foi desenvolvido com HTML, CSS e JavaScript para o front-end, já no Backend foi utilizado o Javascript/Node.js e o MySQL para armazenar os dados,tambem foi utilizado o Express.js como framework para facilitar a criação de API.

Funcionalidades

  • FUNCIONALIDADE 1: ✅Adicionar um novo produto;
  • FUNCIONALIDADE 2: ✅Editar e excluir o produto;
  • FUNCIONALIDADE 3: ✅Exportar os dados para Excel;
  • FUNCIONALIDADE 4: ✅Se a quantidade do produto estiver abaixo de 5 o sistema informa que o estoque está baixo;
  • FUNCIONALIDADE 5: ✅Você pode buscar por produtos se desejar;
  • FUNCIONALIDADE 6: ✅Função de visualizar as informações do produto;
  • FUNCIONALIDADE 7: ✅Botão para descartar produtos.

Desafios e Soluções

O grande desafio foi criar um sistema de gerenciamento de estoque que fosse fácil de usar e entender, além de criar uma interface amigável e intuitiva para o usuário.
